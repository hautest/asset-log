# asset-log

## AI 코드 리뷰

### CodeRabbit

이 프로젝트는 PR이 생성되거나 업데이트될 때 CodeRabbit AI가 자동으로 코드 리뷰를 수행합니다.

#### 설정 완료
- CodeRabbit GitHub App 설치 및 레포 연결 완료
- `.coderabbit.yaml` 설정 파일로 한국어 리뷰 및 프로젝트별 규칙 적용

#### 리뷰 내용
- 코드 품질 및 베스트 프랙티스
- CLAUDE.md 코딩 컨벤션 준수 여부
- TypeScript 타입 안정성
- 잠재적 버그 및 보안 이슈

#### 사용 방법
PR을 생성하면 자동으로 CodeRabbit이 리뷰를 수행하고, 리뷰 결과가 PR 코멘트로 작성됩니다.
