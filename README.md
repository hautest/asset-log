# Asset Log 기획서

## 1. 프로젝트 개요

이 어플리케이션은 자신의 "모든" 자산의 흐름을 한 눈에 볼 수 있게 서비스를 제공합니다.

## 2. 해결하고 싶은 문제

- **기존 서비스의 한계**: 은행/증권사 앱은 연동된 계좌의 잔고만 보여줍니다.
- **숨은 자산 누락**: 전세 보증금, 지인에게 빌려준 돈, 부동산, 연동되지 않는 해외 자산 등이 전체 자산 현황에서 빠져 있습니다.
- **번거로운 계산**: 사용자가 이 모든 자산을 합산하려면 머릿속으로 계산하거나 엑셀을 켜야 합니다.
- **흐름 파악 불가**: 현재 총액만 보일 뿐, 작년이나 지난달 대비 자산이 얼마나 늘고 줄었는지 증감 추이를 한눈에 볼 수 없습니다.

## 3. 핵심 기능 (Core Features)

### 3.1 인증 (Authentication)

- **소셜 로그인**: 구글(Google) 계정을 통한 회원가입 및 로그인만 지원합니다.

### 3.2 랜딩 페이지 (Home)

- **서비스 소개**: 로그인을 하지 않은 사용자에게 서비스의 가치와 특징을 소개합니다.
- **접근 제한**: 랜딩 페이지를 제외한 모든 기능은 로그인 후에만 이용 가능합니다.

### 3.3 대시보드 (Dashboard)

- **메인 뷰**: 로그인 후 최초 진입하는 화면입니다.
- **자산 흐름 시각화**: 전체 자산의 증감 추이를 **바 그래프(Bar Chart)**로 보여줍니다.
- **인터랙션**: 차트의 특정 바(Bar)를 클릭하면 해당 월의 상세 페이지로 이동합니다.
- **미입력 상태 표시**: 데이터를 입력하지 않은 달은 차트에 '미입력' 상태로 시각적으로 구분하여 표시합니다.

### 3.4 월 자산 추가 (Add Monthly Asset)

- **입력 기간 제한**: 매월 1일부터 말일까지, 해당 월의 데이터만 입력할 수 있습니다.
- **미입력 처리**: 해당 기간 내에 입력을 하지 않을 경우, 대시보드 차트에 자동으로 미입력 처리됩니다.

### 3.5 월 상세 페이지 (Monthly Detail)

- **상세 조회**: 특정 월의 자산 구성 내역을 자세히 확인합니다.
- **데이터 관리**: 해당 월의 자산 항목을 추가, 수정, 삭제할 수 있습니다.
- **메모 기능**: 해당 월의 특이사항이나 변동 사유를 메모로 남길 수 있습니다.

### 3.6 마이페이지 (My Page)

- **회원 정보**: 내 정보를 확인하고 수정할 수 있습니다.
- **회원 탈퇴**: 서비스 이용을 중단하고 계정을 삭제할 수 있습니다.
- **카테고리 관리**: 나만의 자산 분류(카테고리)를 추가, 수정, 삭제할 수 있습니다.
- **설정**: 이메일 수신 동의 여부 등을 설정합니다.

### 3.7 자산 추가 알림

- **자산 추가 메일 전송**: 매월 1일에 자산 추가 알림 이메일을 전송합니다.
- **리마인드 메일 전송**: 해당 월에 자산 추가가 없을 시 1주일에 한번씩 리마인드 이메일을 전송합니다.
